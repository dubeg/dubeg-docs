============
Repeat lines
============

Reference
---------
	http://design.tutsplus.com/tutorials/quick-tip-how-to-create-a-smooth-navigation-menu-in-adobe-illustrator--vector-4242


Howto
-----
	1. Create a rectangle -OR- line.
	2. Go into 'Isolation mode'.
	3. Selection Effect -> 3D -> 'Distortion & Transform' -> Transform
		: Horizontal ..
		: Vertical ..
		: Copies: x copies
	4. Object -> Expand appearence

	Notes:
		If object has no outline path or fill, there's won't be anything to see.
