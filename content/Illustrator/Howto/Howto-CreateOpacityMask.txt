===================
Create Opacity Mask
===================

Howto
-----
	1. Create object on which you want to have an opacity mask.
	2. Create copy of the object (to use as a mask).
	3. Fill with gradient (black: transparent, white: opaque).
	4. Put mask on top of objects (in Layers).
	5. Select object and mask.
	6. Go in .. Transparency -> 'Make mask'
