=====================
Create Circular Mesh
=====================

Reference
---------
	http://design.tutsplus.com/tutorials/how-to-create-a-radial-mesh-in-illustrator-and-make-a-vector-cd--vector-2655

Howto
-----
	1. Create ellipse
	2. Set fill - gradient
	3. Expand appearance
	4. Remove/Cancel cutting mask
	5. Delete path in group.
	
	There you go.

	Warning: 
		Select points with lasso or rectangle selection with DirectSelectionTool because they aren't all joined.
